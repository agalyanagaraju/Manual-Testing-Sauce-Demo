# Manual Testing Project – Login Module

## Project Title
Manual Testing of Login Functionality (SauceDemo)

## Project Description
This project focuses on manual testing of the login module of a demo web application.  
The objective is to validate different login scenarios such as valid login, invalid login, empty fields, and error messages.

Test artifacts include Test Cases, Test Plan, Bug Report, and Requirement Traceability Matrix (RTM).

## Application Tested
https://www.saucedemo.com/

## Modules Covered
- Login Page

## Test Scenarios
- Valid Login
- Invalid Username
- Invalid Password
- Empty Username
- Empty Password

## Documents Included
- Test Cases (Excel)
- Test Plan (Word)
- Bug Report (Excel)
- Requirement Traceability Matrix – RTM (Excel)
